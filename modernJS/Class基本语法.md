## Class基本语法

在面向对象的编程中,class是用于创建对象的可扩展的程序代码模板,它为对象提供了状态的初始值和行为的实现

### class 语法

### 什么是class

class User{ } 构造实际做了以下:

1. 创建一个名为User的函数,该函数称为类声明的结果
2. 存储类中的方法
